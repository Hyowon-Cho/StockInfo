### StockInfo
#
### General Info
- StockInfo requires at least python 3.7 version.
- StockInfo is still in the development stage. Please use issue or pull request for feedback such as bugs or function requests.
#
### Features

- Symbol
- Company
- primaryExchange
- latestPrice
- latestTime
- week52High
- week52Low
